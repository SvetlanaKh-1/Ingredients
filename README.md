## Задание

Проанализировать программу расчёта распределения ингредиентов на порцию. Программе на вход даётся рецепт с количеством каждого ингредиента, а также количество человек, которые будут в одинаковых порциях его есть. На выходе программа указывает без округлений, какое точное количество каждого ингредиента пришлось на одну порцию еды.

**В результате необходимо:**

Найти дефект в программе, проанализировав код и вывод программы, и оформить баг-репорт с помощью GitHub Issues.

**Результат:** [Баг-репорт](https://github.com/SvetlanaKh-1/Ingredients/issues/1)

Дан код программы:

```java
public class Main {
    public static void main(String[] args) {

        int eaters = 5; // сколько людей будут есть

        int water = 3000; // миллилитров воды
        int potatoes = 5; // картофелин
        int chicken = 6; // куриных бёдер
        int spices = 10; // ложек специй

        System.out.println("Сварили суп. На одного человека вышло:");
        System.out.println((water / eaters) + " миллилитров(а) воды");
        System.out.println((potatoes / eaters) + " картофелин(а/ы)");
        System.out.println((chicken / eaters) + " куриных(ое) бёдер(ро)");
        System.out.println((spices / eaters) + " ложек(ки/ка) специй");

    }
}
```
